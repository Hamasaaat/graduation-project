1. First, checkout the main branch of the repo from:https://github.com/Hamasaaat/graduation-project.git
2. Open VS code and browse to the folder where you checked out the repo and open it.
3. if you don't have vite: 
 npm install vite --save-dev

==========================
4. then to install the dependencies, run these in the terminal:

npm install
npm install tailwindcss @tailwindcss/vite
npm install react-router-dom
npm install formik yup
npm install uuidv4

============================
5. to launch the app, run the following:
npm run dev
